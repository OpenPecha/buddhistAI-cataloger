Annotation Guidelines   
Overview  
This project involves using tools to split any book into its individual sections, with clear and proper labeling of titles and authors. Once completed, this project is important because it allows Tibetan language users to access quality documents quickly and conveniently through AI.

ཀ༽ ལས་ཀ་བྱེད་མཁན་གྱིས་ལག་བསྟར་བྱེད་དགོས་པ་ཁག (Annotator Instructions)

* དཔེ་དེབ་གང་ཞིག་གི་དཀར་ཆག་སྔོན་གྱི་སྡེ་ཚན་གང་ཡོད་སྡེ་ཚན་གཅིག་ཏུ་གཏུབ་སྟེ་Front matter བདམས་ཏེ་ N/A མནན་དགོས།  
* Front Matter:  
  Merge all sections appearing before the table of contents into one chunk, select Front matter, press N/A.  
    
* དཀར་ཆག་ལོགས་སུ་གཏུབ་ནས་TOC བདམས་ཏེ་ N/A མནན་ནས་ཉར་ཚགས་བྱེད་དགོས།  
* TOC: (Table of Contents)  
   Split the table of contents separately, select TOC, press N/A.  
    
* གཞུང་དངོས་ཀྱི་ཆོས་ཚན་རེ་རེ་གཏུབ་ནས་Text བདམས་ཏེ་གཞུང་གི་མཚན་བྱང་ཡིག་རྐྱང་གཞིར་བཟུང་སྟེ་དག་ཆ་མ་ནོར་བ་དང་བར་སྟོང་སོགས་མེད་པར་ Title ནང་བཀོད་དགོས། རྩོམ་པ་པོའི་མཚན་ལེགས་པར་ངོས་འཛིན་གྱིས་ Author Name in the text ནང་ཉར་ཚགས་བྱེད་དགོས།  
* Text: (Main Body)  
  Split each chapter/section of the main text individually, select Text, and: Enter the title in the Title field — based on the exact text of the section heading, with no spelling errors, no extra spaces

          Identify the author correctly and save under Author Name in the   
          text.

དང་པོ། མཚན་བྱང་གི་སྐོར། 

* གཏུབ་ཟིན་པའི་ཆོས་ཚན་གང་རུང་ཞིག་ལ་མཚན་བྱང་མེད་ན་མཇུག་ཏུ་ཡོད་མེད་ཞིབ་གཟིགས་ཀྱིས་མཇུག་ཏུ་གསལ་ན་དེ་མཚན་བྱང་དུ་བཀོད་དགོས།  
* If a split text section has no title:

          Check the end of the section — if a title appears there, use it as   
          the title.

* མཚན་བྱང་མེད་པ་གང་ཞིག་ལ་མཇུག་ཏུའང་མི་གསལ་ན་སྐུལ་བ་པོ་དང་དུས་ཚོད། ས་གནས་སོགས་ཡོད་ན་དེ་དག་དང་སྦྱར་ནས་མིང་གསར་པ་བཟོ་དགོས། དཔེར་ན་བཀྲ་ཤིས་ཀྱིས་བསྐུལ་ངོར་མཛད་པའོ།། ལྟ་བུ་གསལ་ན་སྡེ་ཚན་བསྟོད་པ་དང་། སྨོན་ལམ་སོགས་གང་ཡིན་དང་སྦྱར་ཏེ་བཀྲ་ཤིས་ཀྱིས་བསྐུལ་ངོར་སྦྱར་བའི་བསྟོད་པ། ལྟ་བུ་བཟོ་དགོས།  
* If there's no title anywhere in the section:

           If there is a requester’s name, date, and location mentioned,      
          combine them to create a new title.   
           Example: If it says "composed at the request of Tashi" → create a   
           title like: "Praise composed at the request of Tashi"

* གོང་གི་དེ་དག་གང་ཡང་མི་གསལ་ན་སྡེ་ཚན་དང་དོན་དུ་གཞིགས་ཏེ་བཅད་ཟིན་པའི་ཆོས་ཚན་གྱི་དབུའི་ཚེག་ཁྱིམ་གཉིས་གསུམ་གང་རུང་དང་སྦྱར་ཏེ་འབྲི་དགོས། དཔེར་ན། ཞབས་བརྟན་རབ་འབྱམས་མ། ལྟ་བུ་བྲིས་ཆོག  
* If none of the above are available:  
  Based on the  meaning of text, take the first 2–3 syllables of the section text, use that as the title. Example:“ཞབས་བརྟན་རབ་འབྱམས་མ།”  
    
* གལ་སྲིད་ཆོས་ཚན་གྱི་མཚན་བྱང་དེ་གཞུང་གི་ཡིག་རྐྱང་དུ་མི་གསལ་བར་དེབ་ཀྱི་རྒྱབ་ཤོག་དང་བདམས་ཟིན་པའི་Front matter ནང་དུ་གསལ་ན་མཚན་བྱང་དེ་བྲིས་ནས་ reconstructed འདི་ལ་འགྲིག་རྟགས་བཀོད་དགོས།  
* If the title is not in the main text but appears in the book's  cover or previously selected Front matter:

          Write that title and mark the "reconstructed" checkbox.

* གལ་ཏེ་དཀར་ཆག་གི་རྗེས་དང། ཆོས་ཚན་གང་ཞིག་གི་མགོ་མཇུག་གཉིས་སུ་མཛད་པ་པོའི་ངོ་སྤྲོད་མདོར་བསྡུས་དང་། བསྡུས་དོན། དཔར་སྐྲུན་གླེང་བརྗོད་སོགས་གསལ་ན་མཛད་པ་པོའི་ཚབ་ཏུ་མཚན་དེ་ཉིད་སྦྱར་ཏེ་འབྲི་དགོས། དཔེར་ན་ཆོས་ཚན་དེའི་མཛད་པ་པོ་རྒྱལ་སྲས་ཞི་བ་ལྷ་ཡིན་ན། རྒྱལ་སྲས་ཞི་བ་ལྷ་ངོ་སྤྲོད་མདོར་བསྡུས། ལྟ་བུ་དང་། དེབ་ཀྱི་མཚན་བྱང་དུ་བྱང་ཆུབ་སེམས་དཔའི་སྤྱོད་པ་ལ་འཇུག་པ་ཡིན་ན་བྱང་ཆུབ་སེམས་དཔའི་སྤྱོད་པ་ལ་འཇུག་པའི་བསྡུས་དོན་ལྟ་བུ་སྦྱར་ནས་འབྲི་དགོས།  
* If there is a brief author introduction, summary, or publishing note at the beginning/end of a section or after the TOC:  
  Combine the author's name with the section type to form the title.  
  Example: If the author is (རྒྱལ་སྲས་ཞི་བ་ལྷ།) → write (རྒྱལ་སྲས་ཞི་བ་ལྷ་ངོ་སྤྲོད་མདོར་བསྡུས།)    
  If the book title is (བྱང་ཆུབ་སེམས་དཔའི་སྤྱོད་པ་ལ་འཇུག་པ། )→ write (བྱང་ཆུབ་སེམས་དཔའི་སྤྱོད་པ་ལ་འཇུག་པའི་བསྡུས་དོན།)  
* མཚན་བྱང་གོང་བཀོད་དེ་དག་ལྟར་བཟོས་རྗེས་ངེས་པར་དུ་reconstructed འདི་ལ་འགྲིག་རྟགས་བཀོད་དགོས། འགྲིག་རྟགས་བཀོད་པའི་དོན་ནི་བཟོ་བཞིན་པའི་གཞུང་གི་ཡིག་རྐྱང་དུ་མི་གསལ་བ་དང་མཛད་པ་པོའི་བཏགས་པ་མ་ཡིན་པར་ཁྱོད་ཀྱིས་མཚན་བྱང་གསར་དུ་བཟོས་པའི་དོན་ཡིན།  
* Important: Whenever you construct/create a title (not directly from the text), you must check the "reconstructed" box. This indicates the title was not in the original text and was created by you.   
* ཆོས་ཚན་རེ་རེ་གཏུབ་རྒྱུ་ལས་དེའི་ནང་གི་ལེའུ་དང་། རྟོག་པ། སྐབས། རིམ་པར་ཕྱེ་བ་སོགས་གང་ཡང་གཅོད་མི་དགོས།  
* What to Split and What Not to Split  
  Split only individual texts — do NOT further split chapters, sub-sections, bampos or sub-topics within a text  
* དཀར་ཆག་ཏུ་གསལ་བའི་ཆོས་ཚན་ཁོ་ན་སོ་སོ་གཏུབ་རྒྱུ་ལས་ས་བཅད་སོགས་ལོགས་སུ་གཏུབ་མི་དགོས།  
* Split only the sections listed in the table of contents — do not separately split headings/outlines  
    
* དཀར་ཆག་ཏུ་མི་གསལ་ཡང་གསུང་ཐོར་བུ་ལྟ་བུ་དང་བསྟོད་པ་སོགས་ཕྱོགས་བསྒྲིགས་ལྟ་བུ་ཡིན་ན་དེའི་ནང་གི་ཆོས་ཚན་གྱི་རིང་ཐུང་ལ་མ་ལྟོས་པར་ངེས་པར་གཏུབ་དགོས།  
* If a section is not in the TOC but appears to be a collection (like miscellaneous writings or praises/hymns), split each piece within it individually, regardless of length  
* མཚན་བྱང་གི་མཐར་ཞེས་བྱ་བ་བཞུགས་སོ།། ཅེས་བྱ་བ་བཞུགས་སོ།། བཞུགས་སོ།། ལྟ་བུ་མི་དགོས་པར་མཚན་བྱང་རང་གི་མཇུག་ཏུ་ཤད་ངེས་པར་འགོད་དགོས།  
* Do NOT add endings like "ཞེས་བྱ་བ་བཞུགས་སོ།" or "བཞུགས་སོ" to the title. The title should simply end with a shad (།).  
* མཇུག་ཏུ་དེབ་ཀྱི་དཔར་བྱང་སྨོན་ཚིག་སོགས་གཏུབ་སྟེ་Back matter ནང་བདམས་ཏེ་N/A མནན་ནས་ཉར་ཚགས་བྱེད་དགོས།  
* Back Matter  
  At the end, split the colophon, dedication prayers, etc., select Back matter, press N/A.  
* ཁྱེད་ཀྱིས་གོང་གི་འདི་དག་ལེགས་པར་བཟོས་རྗེས་Submit བྱེད་དགོས།  
* Once all of the above is done, click Submit.  
* གལ་ཏེ་དེབ་ཀྱི་མ་ཡིག་གི་བཤེར་དཔར་དེ་གསལ་པོ་མེད་པ་དང་། ཡིག་གཟུགས་ཀྱི་ཁྱད་པར་སོགས་ཀྱིས་ཡིག་རྐྱང་དེ་ཉིད་ནོར་འཁྲུལ་མང་དྲགས་ཏེ་གཏུབ་རྒྱུ་ཧ་ཅང་དཀའ་ན་ Skip བྱེད་ཆོག  
*  When to Skip  
  If the original scan of the book is unclear, or font variations make the text too difficult to read with too many errors, you may click Skip.
